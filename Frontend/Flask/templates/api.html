Search File Dataset

get /api/v2/filedataset/list

The List File Dataset API allows a user to list the file datasets of the BigDataOcean Harmonization Tool.

Parameters

Name Required Type Description
For example, using the BigDataOcean Harmonization Tool API, your app can list all file datasets in BigDataOcean Harmonizationhttp://localhost:5000/api/v2/filedataset/list

get /api/v2/filedataset/info

The Info File Dataset API allows a user to get details about one Dataset based on its metadata ID over the BigDataOcean Harmonization Tool.

Parameters

Name Required Type Description
For example, using the BigDataOcean Harmonization Tool API, your app get the details about a specific file dataset "MEDSEA_ANALYSIS_FORECAST_WAV_006_011": http://localhost:5000/api/v2/filedataset/info?search=MEDSEA_ANALYSIS_FORECAST_WAV_006_011

post /auth

The Authentication API allows a user to create an authorization code to use the APIs.

Parameters

Name Required Type Description
For example, 
                  curl -X POST \
                  http://localhost:5000/auth \
                  -H 'Content-Type: application/json' \
                  -d '{
                    "username": "put here username",
                    "password": "put here password"
                    }'
